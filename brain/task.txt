   - meta-llama/llama-3.3-8b-instruct:free
   - meta-llama/llama-3.1-70b-instruct:free
   - mistralai/mistral-7b-instruct:free
   - google/gemma-2-9b-it:free
   - deepseek/deepseek-r1:free
   - microsoft/phi-3-mini-128k-instruct:free
   Let user pick with arrow keys (use InquirerPy).

4. Save encrypted to DB config table. Print success.
5. Allow re-running with: brain setup

## 3D VISUALIZATION (viz/index.html)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
This is the crown jewel. Build a stunning 3D brain/neural network visualization.

### Data source
viz/server.py serves GET /api/graph which returns JSON:
{
  nodes: [{id, title, tags, summary, connections_count, created_at}],
  edges: [{source_id, target_id, similarity_score}]
}
Only include edges where similarity_score > 0.45.
Server also serves the static index.html at GET /.
On `brain viz`, start server on port 7007 and open http://localhost:7007 in browser.

### Visual design — make this genuinely beautiful
Use Three.js r128 from CDN (https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js).

Background: deep space black (#050510)

Nodes:
- Render as glowing spheres using MeshPhongMaterial
- Size = log(connections_count + 2) * 4, minimum 5px
- Color by tag cluster: hash the dominant tag to a hue, use HSL
- Nodes with no tags: soft white glow (#aaaacc)
- Add a subtle pulsing animation (scale oscillates ±5% with sine wave, each node offset randomly)
- On hover: node brightens, shows a floating label (note title)

Edges:
- Render as Lines with LineBasicMaterial
- Opacity = similarity_score (so stronger connections are more visible)
- Color: a dim electric blue (#1a3a6a) for weak, bright cyan (#00ffff) for strong (>0.8)
- Only render edges for the selected/hovered node's connections on hover (performance)

Lighting:
- Ambient light: dim (#111133)
- Two PointLights orbiting slowly: one purple (#6600ff), one teal (#00ffaa)
- Add a faint hemisphere light from above

Layout:
- Use a force-directed 3D layout: initialize nodes randomly in a sphere,
  then run 60 iterations of repulsion + spring forces to space them out naturally
- Nodes connected by strong edges pull together; unconnected nodes repel

Camera:
- OrbitControls-style: implement simple mouse drag to orbit, scroll to zoom
- Auto-rotate slowly when idle (0.001 rad/frame)
- Click a node: camera smoothly lerps toward it (TWEEN-style, manual lerp is fine)

UI overlay (HTML over canvas, absolute positioned):
- Top left: "⬡ second brain" in small monospace, dim white
- Top right: stats bar — "{n} notes · {e} connections · {tags} tags"
- Bottom: search bar — type to highlight matching nodes (filter by title/tag)
- Right panel (slides in on node click): shows note title, summary, tags, 
  "Open file" button (sends filepath to server which opens in default editor),
  list of top 3 connected notes as clickable chips
- Color legend at bottom left: maps colors to top 5 tags

Performance: if >200 nodes, use InstancedMesh for nodes instead of individual meshes.

### Aesthetic feel
Think: "your thoughts as a living galaxy". The whole thing should feel alive — 
nodes breathing, lights slowly orbiting, connections shimmering. 
Someone should be able to stare at it and understand the shape of their thinking.
