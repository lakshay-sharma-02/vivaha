# Homepage — Section 03

# How Vivah Works

> Simplicity builds confidence. Users should understand the entire journey in less than thirty seconds.

---

# Purpose

This section transforms trust into understanding.

After believing Vivah is trustworthy,

users now want to understand

"How does it actually work?"

The explanation should remove uncertainty.

Not create complexity.

---

# Emotional Journey

```
Trust

↓

Understanding

↓

Confidence

↓

Readiness
```

---

# Core Principle

The process should feel

Simple.

Human.

Transparent.

Not algorithmic.

Avoid technical explanations.

Never mention

Machine Learning.

Artificial Intelligence.

Matching Algorithms.

Database.

Technology is invisible.

People remain visible.

# Journey Timeline

Exactly five steps.

```
Create Profile

↓

Verification

↓

Discover Profiles

↓

Meaningful Conversations

↓

Build A Relationship
```

Every step occupies equal visual importance.

---

# Step One

## Create Your Profile

Users share

Identity.

Lifestyle.

Values.

Preferences.

Purpose

Help others understand you.

Not impress others.

---

# Step Two

## Verification

Explain

Identity verification.

Profile review.

Privacy.

Safety.

Verification builds confidence before discovery begins.

---

# Step Three

## Discover Profiles

Show

Thoughtfully presented profiles.

Meaningful information.

Shared values.

Never

Swipe mechanics.

Gamification.

Infinite scrolling.

---

# Step Four

## Meaningful Conversations

Explain

Respectful communication.

Secure messaging.

Comfortable introductions.

Conversations should feel intentional.

---

# Step Five

## Build Your Journey

Marriage is the objective.

Not engagement metrics.

The journey continues offline.

Vivah simply creates the beginning.

# Icons

Use one outline icon per step.

Examples

Profile.

Shield.

Compass.

Chat.

Home.

Consistent stroke width.

No decorative illustrations.

---

# Motion

Allowed

Sequential fade.

Timeline reveal.

Small icon animation.

Forbidden

Large movement.

Bounce.

Progress bars.

Animated arrows.

---

# CTA

One button.

Examples

Create Your Profile

Begin Your Journey

---

# Accessibility

Timeline should remain understandable without animation.

Icons require labels.

Logical reading order.

Keyboard navigation.

---

# Performance

Static content.

SVG icons.

Minimal JavaScript.

No unnecessary animations.

---

# AI Implementation Rules

Always

Use five steps.

Keep explanations short.

Use timeline layout.

Never

Invent additional steps.

Mention AI.

Complicate the process.

---

# Review Checklist

□ Can users understand Vivah within 30 seconds?

□ Are there exactly five steps?

□ Does every step feel equally important?

□ Does the journey end with relationships instead of software?

□ Is accessibility complete?

Only if every answer is "Yes" should this section be approved.
